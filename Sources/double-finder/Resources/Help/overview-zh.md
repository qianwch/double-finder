# Double Finder

Double Finder 是一款 macOS 双面板文件管理器，尽量复刻 Total Commander。

## 面板
左右两个独立面板并排。**活动**面板有焦点；按 **Tab** 切换两侧。复制、移动等操作从活动面板指向另一侧。按 **⌘U** 交换两个面板。

## 导航
双击文件夹或按 **回车** 进入；**退格** 回到上级。用路径栏或 **⌘⇧G**（前往文件夹）跳转到任意位置。**⌘L** 聚焦命令行；**⌘⇧T** 在当前目录打开终端。

## 标签页
每个面板有文件夹标签：**⌘T** 新建，**⌘W** 关闭，点击标签切换。

## 选择与过滤
**⌘A** 全选。输入 **+ / - / \*** 按通配符选择、反选、反转。**⌘F** 打开快速过滤，边打字边缩小列表。

## 文件操作（功能键）
**F2** 改名 · **F3** Quick Look · **F4** 编辑 · **F5** 复制 · **F6** 移动 · **F7** 新建文件夹 · **F8** 删除。**⌘⌫** 移到废纸篓（而非永久删除）。

## 压缩包
zip / tar / 7z / rar 等压缩包能像文件夹一样浏览——选中按回车即可。**⌥F5** 把选中项打包到另一面板；**⌥F6** 解压。

## SFTP
**⌘N** 打开 SFTP 连接；远端目录（及其上的压缩包）可像本地一样浏览。

## 查找与同步
**⌘⇧F** 按名称或内容查找文件。Commands 菜单提供目录比较与同步。

## 视图
**⌘1 / ⌘2 / ⌘3** 切换 完整 / 简洁 / 缩略图 视图。**⌘⇧D** 切换目录树；**⌘⇧B** 展平子树（分支视图）。

## 设置与语言
**⌘,** 打开设置。General 页有 **Language** 下拉——Double Finder 内置中、日、英、韩、德、法六种语言，可实时切换。
